# Cat Feeder

## an app to compare cat food by ingredients

### Install

Clone repo then install using `yarn`

```bash
$ yarn install
```

### Usage

Run docker container to run database and node server

```bash
$ docker-compose up
```

Run the app

```bash
$ cd web
$ yarn dev
```

See in browser at `http://localhost:5173`
Test Server at `http://localhost:3000`
